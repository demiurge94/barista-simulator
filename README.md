# Comp 565 Project: First Person Barista Simulator in Unity

## Setup

1. Open the project in **Unity 6** (6000.3.5f2)
2. In the Project panel, navigate to **Assets > Scenes**
3. Open the **Level** scene
4. Press **Play** to start the game

## Controls

- **WASD** — Move
- **Mouse** — Look around
- **Mouse Click** — Interact
